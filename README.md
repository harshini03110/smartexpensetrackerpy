"# smartexpensetrackerpy" 
▶️ How to Run
First, make sure Python is installed on your system.
You can check it by typing:
python --version
If it shows a version, you’re good to go.
Next, install the required library (only needed for the chart part):
pip install matplotlib
Now download this project or clone it using:
git clone https://github.com/harshini03110/smartexpensetrackerpy.git
After that, open the project folder in VS Code or terminal.
Run the program using:
python expense_manager.py
Once it starts, you’ll see a menu.
You can choose options to:
Add expenses
View or delete entries
Check monthly spending
See category summary
Display chart
Just enter the option shown in the menu.
📌 Small Note
All the data will be saved automatically in a JSON file inside the folder.
If the file is not there, the program will create it on its own.
